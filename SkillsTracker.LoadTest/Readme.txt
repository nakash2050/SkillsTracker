************************Load Testing with Apache JMeter*********************************
1. JMeter (https://jmeter.apache.org/) is available under the following path of the Cloud Machine
		C:\Softwares\JMeter
2. Open the "jmeter.bat" under the "bin" folder. JMeter tool opens.
3. Under File -> Open -> (browse to the "SkillsTrackerLoadTest.jmx" Test Plan kept under the "Test Plan" folder"). 
   500 concurrent users for 10 minutes is already set up in the Test Plan.
4. Click on "Start" button and wait for 10 minutes.
5. The "SkillsTracker_LoadTesting_SummaryReport.csv" and "Skills_Tracker_Load_Testing_Screenshots.docx" are available under "Reports" folder.
